## 项目概览：crewboard（神马排班）

首次代码了解，完整分析如下：

### 技术栈
- **后端**：Node.js + Express + SQLite（better-sqlite3）
- **前端**：纯 Vanilla JS SPA，无构建工具，无框架
- **部署**：PM2 + Nginx 反向代理，运行于 resource.skandstudio.com

### 核心功能模块
1. **资源排程**（schedule.js）：周视图/月视图，可视化排程日历，支持批量多人排程
2. **工时表**（timesheets.js）：按人员按周填报，支持批量保存
3. **报表**（reports.js）：利用率报表 + 项目报表，Excel 导出
4. **人员管理**（manage.js）：Resources CRUD，项目/客户管理，存档功能
5. **企业管理**（enterprise.js）：多租户企业系统，成员权限管理，企业 Webhook 通知
6. **认证**（auth.js）：手机/邮箱登录注册，scrypt 密码加密，7天 Session Token

### 数据库（SQLite）11张表
- resources, clients, projects, bookings, timesheets, leave_entries
- enterprises, users, join_requests, sessions, invitations

### 文件结构
- `server.js` - 入口，Express 初始化
- `db/schema.js` - 数据库 schema + 迁移 + 演示数据
- `db/holidays.js` - 中国节假日数据（2025-2026），自动更新
- `routes/api.js` - 核心业务 API（24KB）
- `routes/auth.js` - 认证 & 企业管理 API（19KB）
- `routes/webhook.js` - 钉钉/企微/飞书 Webhook 推送
- `public/index.html` - SPA 单页，8个页面视图
- `public/css/style.css` - CSS（44KB），支持 12 种主题色
- `public/js/core.js` - 全局 state、API helper、导航
- `public/js/schedule.js` - 排程页面（42KB，最大）
- `public/js/manage.js` - 人员/项目/客户管理（37KB）
- `public/js/enterprise.js` - 企业/设置/账号（27KB）
- `public/js/reports.js` - 报表（8KB）
- `public/js/timesheets.js` - 工时表（8KB）

### 特色
- 中国本土化：节假日系统、人民币计费、钉钉/企微/飞书集成
- 多租户数据隔离（enterprise_id）
- 无前端框架，极简 4 个 npm 依赖
- 支持 SEED_DEMO=1 一键生成演示数据
